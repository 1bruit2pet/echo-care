# Mapping des données (Source : MEDISTOCK.xlsm)
L'application doit supporter les champs identifiés dans le fichier source :
- **REF_INTERNE :** Identifiant unique Medistock.
- **DESIGNATION :** Nom du produit.
- **CATEGORIE :** Type (Consommable, Appareillage, etc.).
- **PRIX_U :** Valeur pour calcul de la valeur du stock embarqué.
- **CODE_BARRE :** Champ de recherche principal pour le scanner.
